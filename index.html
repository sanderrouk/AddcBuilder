Addcbuilder

Java application which builds ADDC Servers from Ubuntu servers using Samba 4 and SSH

View on GitHub Download .zip Download .tar.gz

What is ADDC Builder

Addc builder is a Java desktop application which can be run on any computer to configure an Ubuntu server to be an active directory domain controller. All that is required that the machine running the program has JRE8 installed and is able to SSH to the target server either over a local network or over the internet. Additional info can be found on the wiki.

Download

You can find the latest version of the program from the releases page: https://github.com/sandervend/AddcBuilder/releases. Alternatively you could download the source code and build it yourself using the gradle wrapper or by running it using the gradle wrapper.

// For building
graldew :distZip

// For running
gradlew :run

Using the ADDC Builder

In order to run the program the client computer must have JRE8 or later installed. The client computer must be able to SSH into the server(key authentication is not yet supported). The server must have the package krb5-user installed.

A more detailed explanation of how to run the program can be found from the wiki.
